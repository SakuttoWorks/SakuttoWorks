### 👋 サクッと！自動化工房 (Sakutto Works)

**「面倒な作業を、サクッと解決。」**
中小企業・個人事業主・投資家のためのPython自動化ツール開発所です。
「環境構築不要」「黒い画面を見せない」をモットーに、現場で本当に役立つツールを公開しています。
